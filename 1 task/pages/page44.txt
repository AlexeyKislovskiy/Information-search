<p><b>Urum</b> is a <a href="/wiki/Turkic_language" class="mw-redirect" title="Turkic language">Turkic language</a> spoken by several thousand <a href="/wiki/Greeks" title="Greeks">ethnic Greeks</a> who inhabit a few villages in southeastern <a href="/wiki/Ukraine" title="Ukraine">Ukraine</a>. Over the past few generations, there has been a deviation from teaching children Urum to the more common languages of the region, leaving a fairly limited number of new speakers.<sup id="cite_ref-2" class="reference"><a href="#cite_note-2">&#91;2&#93;</a></sup> The Urum language is often considered a variant of <a href="/wiki/Crimean_Tatar_language" title="Crimean Tatar language">Crimean Tatar</a>.<sup class="noprint Inline-Template Template-Fact" style="white-space:nowrap;">&#91;<i><a href="/wiki/Wikipedia:Citation_needed" title="Wikipedia:Citation needed"><span title="This claim needs references to reliable sources. (October 2023)">citation needed</span></a></i>&#93;</sup>
</p>

<h2><span class="mw-headline" id="Name_and_etymology">Name and etymology</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Urum_language&amp;action=edit&amp;section=1" title="Edit section: Name and etymology"><span>edit</span></a><span class="mw-editsection-bracket">]</span></span></h2>
<p>The name <i><a href="/wiki/Urum_(disambiguation)" class="mw-redirect mw-disambig" title="Urum (disambiguation)">Urum</a></i> is derived from <i><a href="/wiki/R%C3%BBm" class="mw-redirect" title="Rûm">Rûm</a></i> ("Rome"), the term for the <a href="/wiki/Byzantine_Empire" title="Byzantine Empire">Byzantine Empire</a> in the Muslim world. The <a href="/wiki/Ottoman_Empire" title="Ottoman Empire">Ottoman Empire</a> used it to describe non-Muslims within the empire. The initial vowel in Urum is <a href="/wiki/Prosthesis_(linguistics)" class="mw-redirect" title="Prosthesis (linguistics)">prosthetic</a>. Turkic languages originally did not have <span class="IPA nowrap" lang="und-Latn-fonipa" title="Representation in the International Phonetic Alphabet (IPA)">/ɾ/</span> in the word-initial position and so in borrowed words, it used to add a vowel before it. The common use of the term <i>Urum</i> appears to have led to some confusion, as most <a href="/wiki/Turkish_language" title="Turkish language">Turkish</a>-speaking Greeks were called Urum. The Turkish-speaking population in <a href="/wiki/Georgia_(country)" title="Georgia (country)">Georgia</a> is often confused with the distinct community in Ukraine.<sup id="cite_ref-3" class="reference"><a href="#cite_note-3">&#91;3&#93;</a></sup><sup id="cite_ref-4" class="reference"><a href="#cite_note-4">&#91;4&#93;</a></sup>
</p>
<h2><span class="mw-headline" id="Classification">Classification</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Urum_language&amp;action=edit&amp;section=2" title="Edit section: Classification"><span>edit</span></a><span class="mw-editsection-bracket">]</span></span></h2>
<p>Urum is a <a href="/wiki/Turkic_language" class="mw-redirect" title="Turkic language">Turkic language</a> belonging to the <a href="/wiki/Kipchak_languages" title="Kipchak languages">Kipchak branch</a> of the family. According to <i><a href="/wiki/Glottolog" title="Glottolog">Glottolog</a></i>, Urum is a West Kipchak language and forms a subfamily with the Crimeaic languages (<a href="/wiki/Crimean_Tatar_language" title="Crimean Tatar language">Crimean Tatar</a> and <a href="/wiki/Krymchak_language" title="Krymchak language">Krymchak</a>).<sup id="cite_ref-5" class="reference"><a href="#cite_note-5">&#91;5&#93;</a></sup>
</p>

<h2><span class="mw-headline" id="Publications">Publications</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Urum_language&amp;action=edit&amp;section=8" title="Edit section: Publications"><span>edit</span></a><span class="mw-editsection-bracket">]</span></span></h2>
<p>Very little has been published on the Urum language. There exists a very small lexicon,<sup id="cite_ref-11" class="reference"><a href="#cite_note-11">&#91;11&#93;</a></sup> and a small description of the language.<sup id="cite_ref-12" class="reference"><a href="#cite_note-12">&#91;12&#93;</a></sup>
For Caucasian Urum, there is a language documentation project that collected a dictionary,<sup id="cite_ref-13" class="reference"><a href="#cite_note-13">&#91;13&#93;</a></sup> a set of grammatically relevant clausal constructions,<sup id="cite_ref-14" class="reference"><a href="#cite_note-14">&#91;14&#93;</a></sup> and a text corpus.<sup id="cite_ref-15" class="reference"><a href="#cite_note-15">&#91;15&#93;</a></sup> The website of the project contains issues about language and history.<sup id="cite_ref-16" class="reference"><a href="#cite_note-16">&#91;16&#93;</a></sup>
</p>
